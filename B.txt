자료 - 전달 할수 있도록 만든것 기호화한것 방대한 데이터
         목적을 위해 모아놓은 데이터
         고객의 주소, 성별, 이름, 나이, 스마트폰 기종, 활용횟수
정보 - 자료를 문제해결에 도움이 되도록 가공한것 방대한 데이터를 잘가공 한것
         자료를 특정한 목적을 달성 하기 위해 필요한 것
         중년층의 스마트폰 기종, 스마트폰 활용 횟수
지식 - 일반화된 정보 기본 상식


정보화 사회 - 인류가 편리하게 살아갈수 있는 사회 또한 어떤한 위험을 대비
                  하거나 더 좋은 미래를 위해 정보화 사회가 필요하다.

기획 - 원하는 목표를 달성하기 위해 필요한 화동에 대해 생각하는 과정
          개념적 기술이 필요한 심리적 측면과 같은 계획의 작성 및 유지 관리

( IT  ) 정보기술
( BT ) 생명공학
( NT ) 나노기술
( ST ) 우주항공기술
( CT ) 문화산업
( ET ) 환경기술

기획 - 수집 - 관리 - 활용

정보의 기획
What 무엇을? 정보의 입수대상을 명확히 한다.
Where 어디에서? 정보의 소스(정보원)을 파악한다.
When 언제까지? 정보의 요구(수집)시점을 고려한다.
why 왜? 정보의 필요목적을 염두에 둔다
who 누가? 정보활동의 주체를 확정한다.
How 어떻게? 정보의 수집방법을 검토한다.
How Much 얼마나? 정보수집의 비용성(효용성)을 중시한다

사이버 성폭력- 채팅이나 게시판을 통해 성적 수치심을 주는 사이버성폭력
사이버 언어폭력-욕설, 비방, 도배, 성적 욕설(음담패설), 유언비어, 악성 댓글
언어 훼손-단순히 재미로 줄여서 쓰는 언어 올바르지 않은 언어
인터넷 중독-인터넷에 빠져 생활 곤란. 온라인게임이나 음란물 몰입
불건전한 교제-채팅이나 메신저를 이용하여 교제
